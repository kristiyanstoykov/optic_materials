\doxysubsection{Detailed Description}
Class representing an order which includes optic materials and a supplier. 

This class encapsulates the details of an order, including a list of optic materials, a supplier, and methods for managing and calculating the total cost of the order. It extends the \doxylink{class_base}{Base} class, providing implementations for serialization and deserialization to/from streams and JSON.
