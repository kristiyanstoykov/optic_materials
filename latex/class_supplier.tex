\doxysubsection{Detailed Description}
Class representing a supplier in the optics materials system. 

This class stores details about a supplier, including identification, contact information, and profit margin. It extends the \doxylink{class_base}{Base} class, providing implementations for serialization and deserialization to/from streams and JSON.
