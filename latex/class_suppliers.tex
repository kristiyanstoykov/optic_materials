\doxysubsection{Detailed Description}
Class representing a collection of \doxylink{class_supplier}{Supplier} objects. 

This class manages a collection of \doxylink{class_supplier}{Supplier} objects and provides methods for adding suppliers and accessing them. It extends the \doxylink{class_base}{Base} class, implementing serialization and deserialization to/from streams and JSON.
